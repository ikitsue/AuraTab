# ✨ AuraTab - Extensión Moderna para Nueva Pestaña

Transforma tu página Nueva Pestaña con una interfaz moderna y rica en características. Rastrear el clima, zonas horarias y personaliza tu experiencia de navegación.

## 📋 Archivos del Proyecto

### Archivos Principales

- **manifest.json** → Configuración del navegador (Manifest V3)
- **background.js** → Service Worker (eventos, almacenamiento)
- **newtab.html** → Interfaz de nueva pestaña
- **newtab.js** → Lógica principal
- **settings.html** → Página de configuración
- **settings.js** → Gestión de configuración
- **i18n.js** → Sistema de traducción
- **translations.json** → Diccionario (5 idiomas)

## ✨ Características Principales

- 🎨 Fondos de pantalla dinámicos personalizados
- 🌍 Soporte de múltiples ubicaciones (45+ ciudades)
- 🌡️ Clima en tiempo real
- 🔍 Búsqueda multi-motor
- 🌐 5 idiomas integrados
- 🔗 Accesos directos personalizables
- ⚙️ Configuración avanzada
- 🔊 Control completo de sonido

## 🚀 Cómo Cargar

1. Abre `chrome://extensions/` o `brave://extensions/`
2. Habilita "Modo de desarrollador"
3. Haz clic en "Cargar extensión sin empaquetar"
4. Selecciona la carpeta AuraTab
5. ¡Listo! ✅
