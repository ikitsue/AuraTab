# ✨ AuraTab - Современное расширение новой вкладки

Преобразуйте вашу страницу новой вкладки с современным и функциональным интерфейсом. Отслеживайте погоду, часовые пояса и настраивайте свой опыт просмотра.

## 📋 Файлы проекта

### 1. **manifest.json**
Конфигурация расширения (требуется Chrome).

**Что он делает:**
- Объявляет имя, версию и описание расширения
- Перечисляет необходимые разрешения (`storage`, `tabs`, `webNavigation`)
- Указывает на service worker (`background.js`)
- Настраивает переопределение новой вкладки (`newtab.html`)

**Ключевой момент:** `manifest_version: 3` = Manifest V3 (современный стандарт)

---

### 2. **background.js**
Service Worker, работающий в фоновом режиме и управляющий логикой.

**Что он делает:**
- 🎵 **Обработчик аудиособытий** → Воспроизводит звуки открытия/закрытия/перезагрузки вкладки
- 💾 **Сохранение параметров** → Использует `chrome.storage.local`
- 📨 **Связь со страницами** → Отвечает на сообщения от popup/newtab

---

### 3. **newtab.html**
Пользовательский интерфейс для страницы новой вкладки.

**Что отображается:**
- ⏰ Текущее время с поддержкой часовых поясов
- 🌡️ Виджет погоды на основе выбранного местоположения
- 🔍 Поисковая строка с несколькими поисковыми системами
- 📍 Селектор местоположения (страна/город)
- ⚙️ Панель параметров

---

### 4. **newtab.js**
Основная логика для страницы новой вкладки.

**Что она делает:**
- 🕐 **Обновляет часы** → Показывает время в выбранном часовом поясе
- 🌦️ **Получает погоду** → Вызывает API Open-Meteo по местоположению
- 🗺️ **Управляет местоположением** → Сопоставляет 45+ городов с часовыми поясами
- 🎨 **Загружает обои** → Отображает пользовательские или стандартные обои
- 🔗 **Управляет ярлыками** → Настраиваемые быстрые ссылки

---

### 5. **i18n.js**
Система интернационализации для управления переводами.

**Поддерживаемые языки:**
- 🇬🇧 English (en)
- 🇫🇷 Français (fr)
- 🇷🇺 Русский (ru)
- 🇪🇸 Español (es)
- 🇨🇳 中文 (zh)

---

## 🚀 Как загрузить расширение

1. Откройте `chrome://extensions/` или `brave://extensions/`
2. Включите режим "Разработчик"
3. Нажмите на "Загрузить распакованное расширение"
4. Выберите папку AuraTab
5. Готово! ✅
