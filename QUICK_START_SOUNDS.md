# 🎵 Démarrage rapide - Sons personnalisés

Vous venez de terminer l'installation d'AuraTab avec support des sons personnalisés. Voici comment les utiliser en 5 minutes ! ⏱️

## 🚀 5 étapes pour commencer

### Étape 1️⃣ : Ouvrir un nouvel onglet
- Appuyez sur **Ctrl+T** (Windows/Linux) ou **Cmd+T** (Mac)
- Vous verrez votre page AuraTab personnalisée

### Étape 2️⃣ : Accéder aux paramètres
- Cherchez l'icône ⚙️ **en bas à droite**
- Cliquez dessus
- Les paramètres complets s'ouvriront dans un modal

### Étape 3️⃣ : Descendre jusqu'aux sons personnalisés
- Scrollez jusqu'à la section **\"🎵 Sons personnalisés\"**
- Vous verrez 3 sections pour vos sons :
  - 📂 Son ouverture onglet
  - 📂 Son fermeture onglet
  - 📂 Son rechargement page

### Étape 4️⃣ : Télécharger votre premier son
1. Préparez un fichier audio :
   - **Format** : MP3, WAV, OGG, ou M4A
   - **Taille** : Moins de 2MB
   - **Durée** : 0.5-1 seconde idéalement

2. Cliquez sur **\"📤 Charger un son\"** pour la catégorie que vous voulez
3. Sélectionnez votre fichier
4. Attendez la notification ✅ **\"Son personnalisé!\"**
5. Le statut changera en **\"✅ Personnalisé\"**

### Étape 5️⃣ : Tester vos sons
- Scroller un peu vers le bas
- Cliquez sur **\"🔉 Tester les sons\"**
- Vous écouterez l'ordre complet :
  1. Ouverture onglet
  2. Fermeture onglet
  3. Rechargement

✅ **Vous avez terminé ! Vos sons seront joués automatiquement.**

## 💡 Conseils rapides

### Où trouver des sons ?
- **Gratuits & Libres** : 
  - [Freesound.org](https://freesound.org)
  - [Pixabay Sound](https://pixabay.com/sound)
  - [Zapsplat](https://www.zapsplat.com)
  
### Quels sons choisir ?
| Son | Type | Exemples |
|-----|------|----------|
| **Ouverture** | Léger, positif | \"whoosh\", \"ding\", \"pop\" |
| **Fermeture** | Léger, fermant | \"slide\", \"close\", \"whoosh down\" |
| **Rechargement** | Dynamique | \"loading\", \"refresh\", \"notification\" |

### Trop fort ? Trop faible ?
- Allez dans la section "🔊 Effets sonores"
- Réglez le slider **Volume** (0-100%)
- Testez à nouveau avec "🔉 Tester les sons"

## 🆘 Ça ne marche pas ?

### Le son ne se joue pas
1. ✅ Vérifiez que **\"État du son\"** est **\"Activé\"**
2. ✅ Vérifiez que le son spécifique est **coché** (✓)
3. ✅ Vérifiez le volume (pas à 0%)
4. Testez avec "🔉 Tester les sons"

### Erreur \"Format non supporté\"
- Votre fichier n'est pas en MP3, WAV, OGG ou M4A
- Convertissez-le avec [Audacity](https://www.audacityteam.org/) (gratuit)

### Erreur \"Fichier trop volumineux\"
- Votre fichier dépasse 2MB
- Compressez-le avec [Online Convert](https://online-convert.com/)

### Les sons ne se sauvegardent pas
- Videz le cache : Ctrl+Shift+Delete
- Réinstallez l'extension
- Essayez à nouveau
