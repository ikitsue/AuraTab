🚀 GUIDE INSTALLATION RAPIDE

1️⃣ OUVRIR LE GESTIONNAIRE D'EXTENSIONS:

   Google Chrome:  chrome://extensions
   Brave Browser:  brave://extensions
   Microsoft Edge: edge://extensions

2️⃣ ACTIVER LE MODE DÉVELOPPEUR:

   Cliquez le bouton "Mode développeur" en haut à droite

3️⃣ CHARGER L'EXTENSION:

   Cliquez "Charger l'extension non empaquetée"
   Sélectionnez le dossier: AuraTab
   L'extension s'affiche dans la liste ✓

4️⃣ VÉRIFIER L'INSTALLATION:

   ✓ Icon AuraTab visible dans la barre d'outils
   ✓ Ouvrir nouvel onglet (Ctrl+T) → Interface AuraTab
   ✓ Cliquer icon → Popup configuration s'ouvre

5️⃣ UTILISER L'EXTENSION:

   - Paramètres: Cliquez ⚙️ en bas à droite
   - Fond d'écran: Section 🎨 Fond d'écran
   - Sons: Section 🔊 Effets sonores
   - Toutes les modifications sauvegardées automatiquement
