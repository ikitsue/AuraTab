## 🎼 Formats supportés

AuraTab accepte les formats audio suivants :
- **MP3** (.mp3) - Format compressé le plus courant
- **WAV** (.wav) - Format non compressé haute qualité
- **OGG** (.ogg) - Format compressé libre
- **M4A** (.m4a) - Format Apple/iTunes

## ⚠️ Limites et contraintes

| Paramètre | Valeur |
|-----------|--------|
| **Taille maximale par son** | 2 MB |
| **Nombre de sons personnalisés** | 3 (ouverture, fermeture, rechargement) |
| **Durée recommandée** | 0.5 à 2 secondes |
| **Anti-spam cooldown** | 500ms entre deux sons identiques |

## 🎧 Comment télécharger des sons

### Méthode 1 : Via la page de nouvel onglet (recommandée)

1. **Ouvrir un nouvel onglet** → Votre page AuraTab personnalisée s'affiche
2. **Cliquer sur l'icône ⚙️** (en bas à droite) pour ouvrir les paramètres
3. **Aller à la section "🎵 Sons personnalisés"**
4. **Pour chaque son** :
   - Cliquez sur le bouton "📤 Charger un son"
   - Sélectionnez votre fichier audio
   - Le statut changera en "✅ Personnalisé"

### Méthode 2 : Via la popup rapide

1. **Cliquer sur l'icône AuraTab** (barre de navigation)
2. **Aller à la section "🔊 Effets sonores"**
3. **Cliquer sur "🔊 Tester les sons"** pour entendre vos sons actuels

## 🔊 Tester vos sons

Deux façons de tester :

### Option 1 : Bouton dans la page nouvel onglet
```
Paramètres → 🎵 Sons personnalisés → Cliquer "Tester les sons"
```

### Option 2 : Bouton dans la popup
```
Cliquer sur AuraTab → 🔊 Tester les sons
```

Les sons seront joués dans cet ordre :
1. Son d'ouverture onglet (+ 200ms)
2. Son de fermeture onglet (+ 800ms)
3. Son de rechargement (+ 1400ms)

**Note** : Seuls les sons activés seront joués !

## 🔄 Réinitialiser les sons

Pour revenir aux sons par défaut :

1. **Ouvrir un nouvel onglet**
2. **Aller aux paramètres → 🎵 Sons personnalisés**
3. **Cliquer sur le bouton "🔄 Réinitialiser"** à côté du son concerné
4. Le statut repassera à "Par défaut"

## 🔧 Dépannage

### ❌ Le son ne se joue pas

**Causes possibles** :
- Votre navigateur ou onglet a mute le son
- La taille du fichier dépasse 2MB
- Le format n'est pas supporté
- Le son est bloqué par la protection anti-spam (500ms)

**Solutions** :
1. Vérifiez que le son est activé (État du son = "Activé")
2. Vérifiez les paramètres de volume (0-100%)
3. Testez avec un autre fichier audio
4. Ouvrez la console (F12) pour voir les erreurs

### ❌ Erreur "Format non supporté"

Votre fichier n'est pas dans un format accepté. Convertissez-le en MP3, WAV, OGG ou M4A.

Outils gratuits :
- **Audacity** (gratuit, open-source) : audacityteam.org
- **Online Convert** : online-convert.com
- **FFmpeg** (ligne de commande) : ffmpeg.org

### ❌ Erreur "Fichier trop volumineux"

Votre son dépasse 2MB. Vous devez le compresser.

**Astuce** : Pour un son de 1-2 secondes, une qualité de 128kbps en MP3 suffit.

### 💾 Mes sons personnalisés ne se sauvegardent pas

**Causes possibles** :
- Extension sans accès au chrome.storage
- Cache navigateur corrompu
- Réinstallation de l'extension

**Solutions** :
1. Réinstallez l'extension
2. Videz le cache (Ctrl+Shift+Delete)
3. Réessayez de télécharger vos sons

## 📝 Conseils d'utilisation

### Sons recommandés

| Son | Durée | Type recommandé |
|-----|-------|-----------------|
| Ouverture onglet | 0.3-0.8s | Son léger, discret |
| Fermeture onglet | 0.3-0.8s | Son léger, fermant |
| Rechargement | 0.5-1.5s | Son de chargement |

### Où trouver des sons ?

- **Freesound** : freesound.org (CC Libre)
- **Pixabay** : pixabay.com/sound/ (Libre de droits)
- **Zapsplat** : zapsplat.com (Gratuit)
- **YouTube Audio Library** : Recherche sons sur YouTube

## 🚀 Points clés à retenir

✅ Réinitialisation possible à tout moment
