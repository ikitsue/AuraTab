1️⃣  INSTALLATION (5 minutes):

    1. Ouvrir chrome://extensions
    2. Activer "Mode développeur"
    3. Cliquer "Charger l'extension non empaquetée"
    4. Sélectionner le dossier AuraTab
    5. ✓ Extension installée!

    Voir: QUICK_START.md pour détails

2️⃣  VÉRIFICATION:

    □ Ouvrir nouvel onglet (Ctrl+T)
    □ Interface AuraTab s'affiche
    □ Cliquer icon → Popup s'ouvre
    □ Paramètres sauvegardés automatiquement

3️⃣  PERSONNALISATION:

    □ Charger un fond d'écran (paramètres)
    □ Régler le volume
    □ Activer/désactiver les sons
    □ Configurer les options

4️⃣  AMÉLIORATION (Optionnel):

    □ Remplacer fichiers MP3 placeholder
    □ Optimiser les icônes
    □ Modifier les couleurs
    □ Ajouter des raccourcis

5️⃣  DISTRIBUTION (Optionnel):

    □ Publier sur Chrome Web Store
    □ Partager l'extension
    □ Créer repository GitHub
